## ğŸ” **Overview**

The **Student Result Management System (SRMS)** is a web-based application designed to simplify how educational institutes manage student records, exams, marks, and results.
It automates result processing, reduces manual errors, and ensures accuracy and transparency.

Teachers can add marks, admins can manage subjects & exams, and students can instantly view their results online using their Registration Number.
